# **App Name**: Cake-O-licious

## Core Features:

- Menu Display: Displays a menu of cakes, brownies, muffins, croissants, danishes and gift items with images, names, descriptions, and prices.
- Item Selection: Enables users to select from the available bakery items and gift items for purchase
- Item Customization: Presents options to customize bakery items with choices of size, beverage, and add-ons.
- Shopping Cart: Displays selected items with customization details and prices in a shopping cart.
- Quantity Adjustment: Allows users to adjust the quantity of items in the cart.
- Order via WhatsApp: Generates a pre-filled WhatsApp message with order details for easy sharing and ordering.

## Style Guidelines:

- Primary color: Pale rose (#F9DED9) to create a soft, inviting and appetite-friendly environment. The hue evokes warmth, sweetness, and comfort, aligning with baked goods.
- Background color: Very pale rose (#FEF4F3) to maintain consistency, as well as reinforce softness without distracting from content.
- Accent color: Salmon (#F0A98F) to draw the eye, while staying close to the primary color on the wheel.
- Body and headline font: 'PT Sans', sans-serif, a font combining modernity and a touch of warmth for headlines and body.
- Use flat, minimalist icons in a style consistent with the overall design.  Icons should be related to the action (e.g. a trashcan for removing an item).
- Employ a clean, card-based layout for menu items and cart items to create a visually appealing and easy-to-navigate interface.
- Incorporate subtle transitions and animations, such as fading effects or slide-in animations, when updating the cart or switching views, to enhance user engagement without overwhelming.
